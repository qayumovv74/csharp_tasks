# Практические задания 5.1 - 5.10 (C#)

Каждая папка `5_N_Имя/` содержит самостоятельную программу-консольное приложение для одного задания.

## Структура каждой папки

- `Program.cs` — исходный код программы
- `Program.csproj` — файл проекта для .NET 8

## Как запустить (рекомендуемый способ — VS Code)

1. Установи .NET SDK (8 или новее): https://dotnet.microsoft.com/download
2. Установи в VS Code расширение **C# Dev Kit** от Microsoft
3. Открой нужную папку в VS Code: `File → Open Folder → 5_1_Library` (или любую другую)
4. В терминале VS Code (Ctrl+`) запусти:
   ```
   dotnet run
   ```

## Альтернативный способ (Visual Studio)

1. Открой `Program.csproj` двойным кликом
2. Нажми F5 или зелёную кнопку «Start»

## Альтернативный способ (только csc.exe из .NET Framework)

Из методички (раздел 3) — через `csc Program.cs`. Работает только на Windows и для старых версий C#. Если будешь собирать так — некоторые конструкции (например, `int?` nullable в задании 5.10) могут потребовать правок. Лучше использовать `dotnet run`.

## Что в каждой программе

Каждая программа реализует требования соответствующего задания и содержит:
- Классы для всех сущностей задания (с private-полями и public-свойствами)
- Словари (Dictionary) и списки (List) для хранения данных
- Тестовые данные, которые загружаются при старте
- Консольное меню, через которое можно выполнить все требуемые операции
- Все методы — статические в основном классе (как в примере с банком из методички)
